# telecom-churn-prediction-
**Project Overview**:
This project is a Machine Learning-based web application that predicts whether a telecom customer will churn (leave the service) or stay based on their usage and account details.
The system is built using a trained ML model and deployed using a Streamlit web application.

**Objective:**
The main goal of this project is to:
Analyze telecom customer data
Identify patterns that lead to customer churn
Build a predictive model to forecast churn
Help telecom companies improve customer retention

**Machine Learning Model:**
Algorithm used: Random Forest Classifier / Logistic Regression (mention yours)
Trained on telecom customer dataset
Output: Churn (Yes/No) prediction

**Dataset:**
Dataset used: Telco Customer Churn Dataset
Contains customer details like:
Tenure
Monthly Charges
Contract type
Internet service
Payment method

**Technologies Used:**
Python 
Pandas
NumPy
Scikit-learn
Matplotlib & Seaborn
Streamlit (for web app)
Pickle (for saving model)

**Features:**
Data preprocessing and cleaning
Exploratory Data Analysis (EDA) with graphs
Machine Learning model training
Customer churn prediction system
Interactive Streamlit web interface

**Benefits of This Project:**
Helps telecom companies reduce customer loss (churn)
Improves business revenue by retaining customers
Identifies high-risk customers early
Uses AI/ML to make data-driven decisions
Saves time compared to manual analysis
Improves customer satisfaction and retention strategies
Helps understand customer behavior patterns

**Project Files:**
app.py → Streamlit web application
model.pkl → Trained ML model
WA_Fn-UseC_-Telco-Customer-Churn.csv → Dataset
Telecom_Churn_Analysis.ipynb → Full analysis + graphs
requirements.txt → Required libraries

**How to Run the Project**:
1. Install dependencies
pip install -r requirements.txt
2. Run Streamlit app
streamlit run app.py
3. Results

The model successfully predicts customer churn based on input features and helps identify high-risk customers.
